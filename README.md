## 🎮 游戏介绍
**Geometry Strategy（几何策略生存 - Boss版）**  
这是一个纯前端（HTML + CSS + JavaScript）的几何生存类小游戏，你需要在不断缩小的地图中躲避敌人、击杀方块、三角形、坦克，并在每一关完成后选择升级。

游戏包含：

- 自动瞄准 & 攻击系统  
- 地图逐步缩圈机制  
- 各种不同机制的敌人（含 Boss）  
- 关卡奖励系统  
- 射程、伤害、射速三维升级  
- 金钱掉落 & 自动吸取  
- 粒子与光效  

可直接在线游玩，无需安装。
